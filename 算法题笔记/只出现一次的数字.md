# 只出现一次的数字
给定一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

说明：

你的算法应该具有线性时间复杂度。 你可以不使用额外空间来实现吗？

示例 1:

输入:

```
 [2,2,1]
```

输出: 

```
1
```

示例 2:

输入: 

```
[4,1,2,1,2]
```

输出: 

```
4
```

这题本质上不难，但是因为从来没有考虑过用“异或”这种运算，所以特此记录一下

```c++
class Solution {
public:
    int singleNumber(vector<int>& nums) {
        int i = 0,j=0;
        for (i = 0; i < nums.size(); i++) {
            j=j^nums[i];
        }
        return j;
    }
};
```
